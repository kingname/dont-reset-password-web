# Don't Reset Password — Landing Page

Landing page for the **Don't Reset Password** Chrome extension, which shows password composition rules next to password fields so you never have to reset your password again.

Live site: [dont-reset-password.vercel.app](https://dont-reset-password.vercel.app)

## Tech Stack

- **Next.js** (App Router)
- **Tailwind CSS** + **shadcn/ui**
- **Framer Motion** for animations
- **Bilingual** — English / 中文 toggle

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

## Deploy

Deployed to [Vercel](https://vercel.com). Push to `main` triggers auto-deploy.
